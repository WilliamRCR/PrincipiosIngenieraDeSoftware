Este repositorio contiene ejemplos básicos en Java que ilustran tres principios fundamentales del desarrollo de software:

- DRY (Don't Repeat Yourself)
  Evita duplicar lógica en el código. En lugar de repetir, reutiliza mediante métodos o clases.
  
- KISS (Keep It Simple, Stupid)
  El código debe ser simple, claro y fácil de entender.
  
- YAGNI (You Aren’t Gonna Need It)
  No implementes funcionalidades hasta que sean realmente necesarias.
